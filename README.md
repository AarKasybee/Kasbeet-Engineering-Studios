# Kasbeet Engineering Studios

Advancing applied engineering, systems design, and engineering education.

## Overview
Kasbeet Engineering Studios is dedicated to excellence in engineering practices, innovative systems design, and comprehensive engineering education. Our website showcases our commitment to professional, minimalist design and clean code principles.

## Features
- **Semantic HTML5**: Proper use of semantic elements for accessibility and SEO
- **Professional Design**: Minimalist, engineering-focused aesthetic
- **Clean Code**: Well-structured, maintainable JavaScript with comprehensive documentation
- **Responsive Design**: Mobile-first approach that works on all devices
- **Accessibility**: WCAG-compliant with ARIA labels and keyboard navigation
- **Code Standards**: Comprehensive code quality configuration and documentation

## Project Structure
```
/
├── index.html          # Main website page with semantic HTML5
├── styles.css          # Professional, minimalist stylesheet
├── main.js             # Clean, well-documented JavaScript
├── .editorconfig       # Editor configuration for consistent formatting
├── eslint.config.js    # JavaScript linting rules
├── CODE_STANDARDS.md   # Comprehensive code standards documentation
└── README.md           # This file
```

## Getting Started

### Viewing the Website
Simply open `index.html` in a web browser to view the website locally.

### Development
1. Clone the repository
2. Open the project in your preferred code editor
3. Make changes following the code standards in `CODE_STANDARDS.md`
4. Test in multiple browsers for compatibility

## Code Standards
All code follows strict standards for quality and maintainability:
- Clean, readable code with proper documentation
- Semantic HTML5 for better accessibility and SEO
- Professional, minimalist CSS design
- Modern JavaScript (ES6+) with proper structure
- Comprehensive accessibility features

For detailed code standards, see [CODE_STANDARDS.md](CODE_STANDARDS.md).
